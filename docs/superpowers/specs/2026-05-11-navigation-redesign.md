# GameGuide 导航栏重新设计

## 概述
为 GameGuide 网站设计并实现全站统一的顶部固定毛玻璃导航栏。

## 导航结构

### 导航项
| 页面 | 图标 | 路径 |
|------|------|------|
| 首页 | 🏠 | `/{locale}` |
| 资讯 | 📰 | `/{locale}/news` |
| 折扣 | 🏷️ | `/{locale}/deals` |

### 视觉设计

#### 毛玻璃效果
- 背景: `rgba(255, 255, 255, 0.1)`
- `backdrop-filter: blur(12px)`
- 底部边框: `1px solid rgba(255, 255, 255, 0.2)`

#### 导航项
- 左侧图标 + 右侧文字
- 间距: 每项 24px
- Hover: 背景 `rgba(255, 255, 255, 0.1)`
- 选中态: 背景 `rgba(255, 255, 255, 0.2)` + 文字加粗

#### 语言切换
- 位置: 导航栏最右侧
- 样式: 胶囊按钮，`EN` / `中` 文字
- 点击切换 locale 并保持当前路径

## 布局

```
┌──────────────────────────────────────────────────────────────┐
│  🎮 GameGuide    [🏠 首页]  [📰 资讯]  [🏷️ 折扣]   [EN]  │
└──────────────────────────────────────────────────────────────┘
```

## 组件

### SiteHeader
新建 `app/components/SiteHeader.tsx`

Props:
- `locale: string` - 当前语言
- `currentPath: string` - 当前路径，用于高亮当前页

### 样式
- 固定定位 `position: fixed; top: 0; left: 0; right: 0`
- 高度 60px
- 内边距水平 24px
- 毛玻璃背景
- z-index 确保在最顶层

## 实施步骤

1. 创建 `SiteHeader.tsx` 组件
2. 在 `layout.tsx` 中引入组件
3. 添加毛玻璃样式到 `globals.css`
4. 更新移动端响应式样式

## 技术说明

- 使用 `next/link` 实现客户端导航
- 使用 `usePathname()` 判断当前激活状态
- 语言切换逻辑复用现有的 `LanguageToggle` 逻辑
